## Developement

Run the following commands for running server (BackEnd)

```sh
$> cd server
$> mvn clean install
$> mvn spring-boot:run
```

![line]

## Tech Stack Used

- Backend Framework: Spring Boot
- Backend Language: Java
- Packaging: Apache Maven
- Containerisation: Docker
- Database: MongoDB
- Backend Hosting: Render
